QUESTION: What will this print?
    class Animal:
        def sound(self):
            raise NotImplementedError
    class Dog(Animal):
        def sound(self):
            return 'Bark!'
    a = Dog()\nprint(a.sound())
ANSWER: Bark!
HINT: Dog overrides Animal method.
CODE: C3
ORDER: 3

# Level 2
QUESTION: What will this print?
    class Animal:
        def speak(self): return '...'
    class Dog(Animal):
        def speak(self): return 'Bark'
    class Cat(Animal):
        def speak(self): return 'Meow'
    animals = [Dog(), Cat()]
    for a in animals:
        print(a.speak())
ANSWER: Bark Meow
HINT: Each class defines its own speak().
CODE: D4
ORDER: 1

QUESTION: Which regex pattern matches any 3-digit number?

QUESTION: What does this print?
    import re
    text = 'Call 911 or 112'
    print(re.findall(r'\\d+', text))
ANSWER: ['911', '112']
HINT: findall returns all digit groups.
CODE: F6
ORDER: 3
